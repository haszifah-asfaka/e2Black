# 🔐 e2Black V2 - Web Cipher Generator

Selamat datang di repository resmi **e2Black V2**! Ini adalah aplikasi web prasaja (sederhana) nggo enkripsi lan dekripsi pesen rahasia, digawe khusus nggo njaga keamanan data sensitif.

## 🚀 Fitur Utama
* **Dynamic Cipher System:** Ngacak pesen nganggo logika XOR lan Dynamic Shift sing njelimet.
* **Super Light-weight:** Desain entheng lan stabil, ramah banget nggo HP spek rendahan.
* **Modern Interface:** Tampilane resik lan kepenak dienggo.
